Explore data on the 16 functions supported by federal spending

This visualization shows how federal funds were spent by budget function, which describes the type of activities carried out, as well as object class, which describes what the government purchased with those funds. Clicking the nodes on each end of the visualization shows the relationship to the opposite side and provides a brief introduction to what programs each budget function or object class includes.
