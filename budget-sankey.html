Connecting Budget Function to Object Class

Exploring “the what” and “the why” of government spending

Do you know what the government spends money on and why? Budget functions are categories that describe the purpose of federal spending, or “the why.” Object classes are categories that describe what goods and services were purchased, or “the what.”

How to use this tool:

Hover over each color bar to highlight how each budget function relates to object class categories, and vice versa.

End point(s) used: TAS Categories

From this endpoint, the fields: obligations_incurred_by_program_object_class_cpe, object_class.major_object_class_name, and treasury_account.budget_function_title are grouped and aggregated.

The twenty official congressional budget functions are rolled up into sixteen budget function categories. Smaller budget functions are combined with others with similar missions. The five major object classes are broken down into eleven more specific classes, to provide descriptive and meaningful information than major object class categories alone. Information on object classes are available here.
